# SQL Practice Project

This folder contains SQL queries written to solve real-world analytical problems using relational databases.

## Key Topics Covered
- Database creation and usage
- Data exploration using SELECT
- Filtering with WHERE conditions
- Aggregations using COUNT, SUM, AVG
- GROUP BY and HAVING clauses
- JOIN operations
- Business-focused analytical queries

## Dataset
- Users data
- Sessions data

## Objective
To practice SQL skills for data analysis and strengthen understanding of business-driven SQL queries.
